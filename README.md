# iPhone Exploratory Data Analysis

- Utilized Pandas to read and manipulate data from CSV files. Loaded all iPhone data into a dataframe, then preformed data cleaning and preprocessing tasks.
- Used Matplotlib to create various bar charts and histograms. Plotted data on ratings for various iPhones, the associated RAM, and number of ratings/reviews given to the specific iPhone.
- Utilized Seaborn to provide a more streamlined interface on the data from the csv files. Created meaningful pair plots and heatmaps to distinguish the ratings/reviews given to which iPhone and how well they performed.
